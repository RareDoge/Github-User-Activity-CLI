In this project, I built a simple command line interface (CLI) to fetch the recent activity
of a GitHub user and display it in the terminal. This project helped me practice programming skills

<h1>skills</h1>
<ul>
  <li>including working with APIs</li>
  <li>handling JSON data</li>
  <li>building a simple CLI application.</li>
</ul>

<h2>Usage</h2>
 <pre>python3 Github-User-Activity-CLI.py <span class="pl-k">&lt;</span>github-username<span class="pl-k">&gt;</span></pre>

 <h2>Example</h2>
  <pre>python3 Github-User-Activity-CLI.py raredoge</pre>

  <h2>Expected Output</h2>
  <pre>
- Raredoge Began a Watch Event on RareDoge/Github-User-Activity-CLI
- Raredoge Opened a new issue in RareDoge/Github-User-Activity-CLI
- Raredoge Created a new repository RareDoge/Github-User-Activity-CLI
- Raredoge Pushed 6 commits in RareDoge/Github-User-Activity-CLI
  </pre>
